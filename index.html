01about

I'm an IT and security engineer based in Helsinki, working at the intersection of infrastructure administration and offensive security. My background spans Microsoft 365 / Entra ID administration, MDM fleet management, and SSO implementation, alongside penetration testing and vulnerability assessment work.

My MSc thesis, "Device-as-Detector: Lightweight Energy-Based Detection of WiFi Deauthentication Attacks on Constrained IoT Devices," explored energy-based side-channel current analysis combined with TinyML to detect attacks on resource-constrained hardware, bridging embedded systems and network security.

I've conducted authorized penetration tests that surfaced critical-severity findings, and I build security tooling on the side, including a self-hosted RAG pipeline for data-sovereign threat analysis and a GitHub Action for automated PR vulnerability scanning.
